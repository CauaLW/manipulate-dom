## :clipboard: Sobre o Projeto

Um projeto extremamente simples para a escrita de um <a href="https://medium.com/@caualoewen1/manipulando-a-dom-usando-javascript-para-p%C3%A1ginas-mais-din%C3%A2micas-328a467b4fb" target="_blank">artigo</a>, ensinando de forma básica como utilizar a DOM para criar e modificar elementos para páginas mais dinâmicas.

## :computer: Tecnologias utilizadas

O projeto foi desenvolvido utilizando as seguintes tecnologias:

- HTML
- CSS
- JS

## :information_source: How To Use

- Clone este repositório: `git clone https://github.com/CauaLW/manipulate-dom`
- Vá até o repositório: `cd manipulate-dom`
